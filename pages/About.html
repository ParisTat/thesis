About the Project

An interactive website designed to simplify and visualize complex astrophysical concepts surrounding black holes. Built as part of my undergraduate thesis.

🚀 Purpose

This website was created as part of my thesis, aiming to provide an interactive and educational experience about black holes. It combines scientific theory with user-friendly navigation and engaging content.

🛠️ Technologies

  • HTML5
  • CSS3 (Custom, Responsive)
  • Vanilla JavaScript

📁 Structure

The website consists of five main sections:
Home, About, Thesis, Quiz and Contact.

🎨 Tools & Credits

Built using Visual Studio Code.
Graphics and assets were generated via Photoshop and AI-based tools.
Scientific content based on academic research in black hole physics.

🎯 Design Approach

The website adopts a minimal, clean aesthetic with an emphasis on readability. Techniques like glassmorphism, smooth transitions, and responsive layouts were applied to enhance the user experience.

🚧 Future Improvements

Possible upgrades include adding more interactive simulations, deeper quizzes with scoring, integration of more dynamic visualizations, and multilingual support.
